# Kiss VPN 0.1.3

Поддержка HWID-протокола.

## Что изменилось

Клиент теперь представляется панели kissmain.ru так же, как Happ / Remnawave-Mobile / современный v2RayTun — отправляет 4 заголовка при каждом запросе подписки:

```
x-hwid:         <стабильный UUID v4, генерится при первом запуске>
x-device-os:    Windows
x-ver-os:       <версия ОС, например 11.0.26200>
x-device-model: <имя компьютера>
```

После этого Kiss VPN отображается в **Мои устройства** на kissmain.ru наравне с мобильными клиентами — раньше там была пустая запись «unknown app».

HWID хранится локально, привязан к установке. Переустановка или удаление данных приложения сбрасывает его — поведение совпадает с мобильными клиентами при очистке app data.

## Обновление

Автообновление подхватит в течение пары часов. Вручную — **Настройки → Обновления → Проверить**.
