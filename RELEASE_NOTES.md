# Kiss VPN 0.1.5

Три темы оформления, сортировка правил и массовый багфикс.

---

## Новое

### 🎨 Три темы оформления

В **Настройки → Внешний вид** теперь доступны три темы:

- **Kiss** (по умолчанию) — фирменная розово-фиолетовая на почти чёрном фоне
- **Тёмная** — нейтральная тёмная с серыми поверхностями и голубым акцентом, без бренд-цветов
- **Светлая** — белые карточки на светло-сером фоне, розовый акцент, тёмный текст

Все 27 файлов UI полностью адаптированы — поверхности, текст, бордеры, акценты, кнопки, переключатели меняются мгновенно при смене темы. Реализовано через `ThemeExtension` (313 ссылок на цвета переведены на контекстные).

### 🔀 Сортировка правил

На вкладке **Режимы → По приложениям** рядом с кнопками «Включить/Выключить все» появились кнопки сортировки:
- **↑ Вкл. сверху** — активные правила наверху, выключенные внизу
- **↓ Выкл. сверху** — наоборот
- Повторное нажатие сбрасывает в исходный порядок. Активная кнопка подсвечивается.

## Исправления ошибок

### Критическое

- **Интернет не ломается при ошибке подключения.** Раньше если VPN падал во время коннекта, системный прокси оставался переключённым на мёртвый порт 7890 — браузеры показывали «нет интернета». Теперь прокси откатывается автоматически.

### Высокие

- **Трей: «Выход» теперь корректно отключает VPN.** Раньше кнопка просто убивала окно, оставляя процессы Mihomo/xray и системный прокси висеть. Теперь — сначала disconnect, потом закрытие.
- **Трей: «Подключиться» реально подключает VPN.** Раньше просто открывало окно. Теперь вызывает toggle.
- **Трей: label обновляется.** «Подключиться» ↔ «Отключиться» в зависимости от состояния VPN.
- **Закрытие окна корректно при выходе.** `dispose()` теперь вызывает `disconnect()` → процессы не остаются висеть.
- **Onboarding больше не ведёт в тупик.** Кнопка «Продолжить» после вставки подписки теперь ведёт на главный экран с навигацией, а не на голую страницу подписки без меню.

### Средние

- **Версия в настройках реальная.** Раньше всегда показывало «0.1.0» — теперь читается из билда.
- **Крестик окна учитывает настройку «Сворачивать в трей».** Если выключена — закрывает приложение с корректным disconnect.
- **Автозапуск работает.** Переключатель «Запускать с Windows» теперь реально пишет/удаляет ключ в реестре `HKCU\...\Run`.
- **Логи переподключаются.** WebSocket логов теперь автоматически переоткрывается через 2 сек после обрыва (при переподключении VPN).
- **Список серверов обновляется после refresh подписки.** Раньше провайдер читал кэш один раз и не перечитывал.
- **Защита от двойного подключения.** Быстрый двойной клик по кнопке больше не запускает два параллельных Mihomo.
- **HelperClient: pending-запросы завершаются при close().** Раньше висели вечно.
- **HelperClient: poller корректно останавливается при разрыве pipe.**
- **WebSocket каналы Mihomo API закрываются при пересоздании.** Больше нет утечки сокетов.
- **HelperClient закрывается при ошибке connect().** Раньше pipe и таймер утекали.

### Низкие

- **Счётчики трафика сбрасываются при переподключении.**
- **Цвета статуса хелпера адаптируются к теме** (были захардкожены на Kiss).
- **Pulse-анимация кнопки подключения** работает только при connecting/connected (раньше крутилась 24/7, жрала CPU).
- **Mesh-фон адаптируется к выбранной теме** (нейтральные блобы в тёмной, розовые в Kiss).
